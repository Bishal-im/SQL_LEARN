# DATA_Cleaning Project

## Dataset
- **File:** layoffs.csv
- **Description:** Contains company layoffs data with columns like company, date, total_laid_off, and percentage_laid_off.
- **Source:** https://www.kaggle.com/datasets/swaptr/layoffs-2022

## Objective
- Clean and transform the raw data to make it usable for analysis.
- Handle missing values, convert percentages, and format dates properly.
- Perform exploratory analysis to summarize key trends in the data.

## Files
- `queries.sql` — SQL queries used for cleaning the dataset.
- `exploratory_analysis.sql` — SQL queries used for exploratory analysis.
- `layoffs.csv` — the dataset used in this project.
